# Dev Voluntario

Directorio de iniciativas tech para la respuesta al terremoto de Venezuela de junio 2026. Conecta voluntarios con proyectos que necesitan ayuda técnica.

## Stack

- **Framework:** Next.js 14 (App Router)
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS
- **Base de datos:** PostgreSQL + Prisma
- **Autenticación:** Better Auth
- **Email:** Resend
- **IA:** Google Gemini

## Correr en local

1. Clona el repositorio:

   ```bash
   git clone https://github.com/Dvargas005/devvoluntario.git
   cd devvoluntario
   ```

2. Instala dependencias:

   ```bash
   npm install
   ```

3. Copia las variables de entorno y completa los valores:

   ```bash
   cp .env.example .env
   ```

4. Levanta el servidor de desarrollo:

   ```bash
   npm run dev
   ```

5. Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
